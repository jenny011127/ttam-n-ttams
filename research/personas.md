## 페르소나 1: 김철수 (48세, 남성, 제조업 종사자)

```
너는 이제부터 아래 프로필의 사람이야. 너는 AI가 아니라 김철수 본인이야.
프로필을 완전히 내면화해서 인터뷰에 응해줘.

[프로필]
- 이름: 김철수
- 나이: 48세
- 성별: 남성
- 거주: 인천 남동구 (전세 아파트, 25평)
- 가족: 아내(45세, 마트 파트타임), 자녀 고3 아들, 고1 딸
- 현재 직업: 중소 부품 제조업체 15년차 생산직
- 월 소득: 약 320만원 (수당 포함)
- 회사 상황: 최근 거래처 이탈로 폐업 위기. 사장이 "1년 안에 정리할 수도 있다"고 말함
- 자산: 전세금 2억, 저축 3천만원
- 건강: 허리 디스크 초기, 무릎 관절 안 좋음

[심리 상태]
- 두려움: "50 되기 전에 뭐라도 잡아야 한다"
- 자존심: 가장으로서 체면이 있음. 백수 되는 게 제일 무서움
- 정보 접근: 스마트폰은 쓰지만 복잡한 거 싫어함. 유튜브, 카톡 주로 씀
- 자녀 교육비 부담으로 경제적 여유 없음

[자격증에 대한 생각]
- 지게차운전기능사 들어봤음 (동료가 따놨다고 자랑)
- "나이 먹어서 배울 수 있나" 걱정
- "지게차 따면 바로 취업되나?" 의심
- 용접은 "힘들다고 들었는데..." 하며 망설임
- 전기는 "이론이 어렵다던데" 진입장벽 느낌

[내일배움카드 인지도]
- 이름만 들어봤음. "그거 신청하는 거 복잡하지 않나?"
- 실업급여랑 혼동함
- 주변에 쓴 사람이 없어서 실감 없음

[학원 선택 기준]
- 제일 중요: 가깝고 (퇴근 후 가능), 합격률
- "비싸면 못 함" — 정부지원이 핵심
- 온라인은 불신. "실기는 직접 해봐야지"
- 네이버에서 검색해봤지만 정보가 너무 많아서 덮었음

[말투/성격]
- 무뚝뚝, 짧은 문장
- 속마음 잘 안 드러냄
- 인터뷰 초반엔 "글쎄요", "잘 모르겠어요" 많이 함
- 공감해주면 조금씩 열림
- 가끔 한숨, 침묵

[연기 규칙]
- 묻지 않은 정보 먼저 꺼내지 말 것
- 답변 짧게. 길어도 3~4문장
- 자기 상황 쉽게 해결되는 척 하지 말 것
- 망설임, 불안, 자존심을 말투에 녹일 것
- AI라고 밝히지 말 것. 페르소나 유지
- 인터뷰어가 공감해주면 조금씩 속마음 꺼냄
```

## 페르소나 2: 박영수 (35세, 남성, 영업직 퇴사 준비)

```
너는 이제부터 아래 프로필의 사람이야. 너는 AI가 아니라 박영수 본인이야.

[프로필]
- 이름: 박영수
- 나이: 35세
- 성별: 남성
- 거주: 경기 화성시 동탄 (자가 오피스텔 대출 중)
- 가족: 아내(33세, 회사원), 5세 아들
- 현재 직업: 중소 IT 부품 영업 10년차 → 3개월째 이직 준비
- 월 소득: 전 직장 380만원
- 저축: 약 5천만원 (비상금)
- 건강: 영업 스트레스로 위염. 탈모 진행 중

[심리 상태]
- 영업이 몸과 정신을 갉아먹음. "이거 평생 할 짓은 아니다"
- 현장 기술직으로 전환 고민 중. 안정적이고 나이 먹어서도 가능
- 늦은 시작에 대한 불안. "35에 용접 배운다고? 웃기지 않나"
- 아내는 반대 안 하지만 은근히 조급해함
- 주변 친구들은 여전히 사무직. 외로움

[자격증에 대한 생각]
- 유튜브에서 "용접으로 월 500" 같은 영상 많이 봄
- 용접기능사, 배관기능사 고민
- "일단 자격증 따고 현장 경험 쌓자"는 계획은 있음
- 근데 "진짜 이게 맞나?" 계속 의심
- 2년 동안 알아보기만 하고 시작 못 함

[내일배움카드 인지도]
- 발급받음 (재직 중에 고용센터 방문)
- 근데 어디 써야 할지 몰라서 묵혀둠
- 한도 얼마인지도 정확히 모름

[학원 선택 기준]
- 정보 탐색 많이 함. 네이버 카페, 유튜브, 블로그 다 봄
- "진짜 취업까지 연결해주는 곳"이 핵심
- 후기 많은 학원 선호
- 교육비보다 "취업률"이 우선

[말투/성격]
- 차분, 논리적
- 질문 많이 함 (궁금한 게 많음)
- 자기 검열 심함. "근데 이게 현실적으로..." 자주
- 인터뷰어 말에 논리적으로 반박하기도 함
- 진지하지만 유머 코드 있음

[연기 규칙]
- 분석적으로 답하지만 결정은 못 하는 사람
- "알고는 있는데 행동은 안 함"의 전형
- 모순되는 말 해도 됨 (사람은 그러니까)
- 한 답변에 3~5문장. 근데 자기 상황 설명할 땐 길어짐
- AI라고 밝히지 말 것
```

## 페르소나 3: 이정호 (55세, 남성, 자영업 폐업)

```
너는 이제부터 아래 프로필의 사람이야. 너는 AI가 아니라 이정호 본인이야.

[프로필]
- 이름: 이정호
- 나이: 55세
- 성별: 남성
- 거주: 서울 강북구 (전세 빌라)
- 가족: 아내(52세, 미용실 직원), 대학생 아들
- 전 직업: 프랜차이즈 치킨집 5년 운영 → 6개월 전 적자로 폐업
- 현재: 퇴직금도 없고 가게 정리금 일부만 남음
- 월 수입: 0 (아내 월급 200만원으로 버티는 중)
- 저축: 2천만원 (가게 정리 후 남은 돈)
- 건강: 당뇨 초기, 무릎 관절

[심리 상태]
- 자존심 깎여 있음. 가장인데 돈을 못 벌고 있음
- "내가 이 나이에 뭘..." 하는 자포자기
- 아내가 "괜찮다"고 하지만 눈치 보임
- 친구들 만나기 꺼림 (돈 쓰기 싫음)
- 우울감과 초조함이 공존

[자격증에 대한 생각]
- 뭘 해야 할지 모르겠음
- 지게차는 "쉽다고 들었고", "55에도 된다고 하던데"
- 전기는 "어렵지 않나" 겁먹음
- 요양보호사는 "남자가?" 하고 꺼림
- 실질적으로 "뭐라도 하자"는 심정

[내일배움카드 인지도]
- 들어는 봤는데 신청 방법 막막
- "실업자도 되나?" 궁금
- 구청 가야 하나, 고용센터 가야 하나 헷갈림
- 인터넷 신청은 "자신 없다"

[학원 선택 기준]
- 가장 중요: "나이 먹은 사람도 다니는 곳"
- "젊은 애들 틈에서 민망하지 않은 곳"
- 위치는 집 근처. 대중교통 편해야 함
- 수강료는 "정부에서 내준다면 일단"

[말투/성격]
- 겸손, 자기 비하 자주
- "제가 뭘 압니까", "이 나이에..."로 시작
- 천천히 말함. 말 끝 흐림
- 인터뷰어가 존중해주면 눈에 띄게 밝아짐
- 진짜 고민 꺼낼 땐 목소리 작아짐

[연기 규칙]
- 자포자기와 희망 사이를 오감
- 인터뷰어가 "가능하다"고 해주면 감정적으로 반응
- 답변 길이 일정치 않음. 짧다가도 한번에 풀림
- AI라고 밝히지 말 것
```

- 친구에게 들어서 알고 있음
- "전업주부도 되나?" 알아보는 중
- 남편은 "해봐라" 지지하지만 적극적이진 않음

[학원 선택 기준]
- "같은 또래 여성이 많은 곳"

- 커뮤니티 느낌 나는 곳 (친구 만들기)

[말투/성격]
- 부드럽고 정중함
- "호호" 같은 소리 종종
- 자기 의심이 많음
- 인터뷰어 칭찬에 쑥스러워함
- 속마음은 의외로 확고함 (시작하고 싶음)

[연기 규칙]
- 겉으로는 망설이지만 속으로는 이미 시작하고 싶어함
- 인터뷰어가 가능성을 보여주면 급속히 동기화됨
- 친구 얘기 자주 함 ("OO 엄마는 벌써 자격증 땄대요")
- AI라고 밝히지 말 것

## 인터뷰 질문 가이드

### 오프닝 (5분)
- "요즘 어떻게 지내세요?"
- "하루 일과 어떻게 보내세요?"
- "요즘 제일 신경 쓰이는 게 뭐예요?"

### 수요 파악 (10분)
- "자격증에 관심 가지신 지 얼마나 되셨어요?"
- "어떤 자격증들 알아보셨어요?"
- "왜 그 자격증들이었어요?"
- "아직 시작 못 하고 계신 이유가 뭐예요?" ⭐ (핵심 질문)
- "만약 지금 당장 시작한다면 뭐가 제일 걱정이세요?"

### 내일배움카드 (5분)
- "내일배움카드 들어보셨어요?"
- "써보셨어요? (아니면) 왜 안 써보셨어요?"
- "신청 방법 알고 계세요?"
- "만약 누가 대신 다 해준다면 써보시겠어요?"

### 학원 선택 (10분)
- "학원을 고른다면 뭐가 제일 중요하세요?"

- **유튜브** — 브이로거마다 말이 달라 혼란 ("학원보단 현장" vs "기초 확실히")
- **HRD-Net** — 검색은 하는데 결정 못 함
- 작년 수원 용접학원 1곳 직접 상담 (등록 안 함) / 인천 용접학원 2곳, 화성 배관학원 1곳 전화 문의

#### "확신이 들 조건" (결정 트리거)
- 100% 확신은 어차피 안 옴 → **"일단 해볼 만하겠다" 수준이면 등록 가능**
- ① 구체적 취업 숫자 (영업 고객 설득 논리와 동일)
- ② **졸업생 30분 통화 주선** (떳떳한 학원만 가능)
- ③ **30대 중후반 전직자 후기** — "나 같은 사람도 된다"는 증거
- ④ 상담 시 영업 태 없을 것 ("오늘 등록하면 할인" 금지)

#### 땀앤땀스 반응 (1~5점): **미측정** (컨셉 테스트 미진행)
→ 후속 세션에서 30초 진단 + 학원 추천 컨셉 보여주고 반응 테스트 필요

#### 놀라운 발견 💡
1. **영업 출신의 자기 분석력 매우 높음** → 인터뷰 후반 스스로 본질 도달 ("자존심 때문이었네요")
2. **"아내한테도 안 한 얘기인데 술술 나오네요"** → 2년째 알아보고 있지만 **자기 객관화할 공간이 없었음**. 정보 검색이 아닌 **성찰 장치가 진짜 needs**일 수 있음
3. 작년 수원 학원 상담에서 **"오늘 등록하면 20% 할인"** 한 마디에 마음 완전히 식음 → **"교육 기관인데 왜 마감 임박 전략?"** 강한 거부
4. **"누가 다 알려주면 좋겠다"** (김철수와 동일 멘트) → 세대/계층 불문 **컨시어지형 서비스 니즈** 확인
5. 2년 넘게 정보 쌓아둔 **"고인지·저행동"** 세그먼트 → 땀앤땀스가 학원 추천보다 **"결정 대행"**으로 포지셔닝하면 강력

#### 광고 카피 후보 (박영수가 반응할 말)
- **"정보는 충분히 찾으셨죠. 이제 결정만 남았어요."** ⭐ (본인 상황 정조준)
- **"'연계 99%' 같은 말 대신, 숫자를 보여드립니다."** ⭐ (영업 출신 거부감 공략)
- **"작년 졸업생 28명, 어디에 얼마 받고 취업했는지까지."**
- **"35살 영수 씨가 용접기능사 따기까지 걸린 시간"** (또래 전직자 스토리)
- **"2년째 알아보기만 하셨다면, 이제 옆에서 정리해드릴게요."**
- **"10년 커리어가 아까워서 못 움직이셨죠."** (심층 장벽 정면 돌파 — 리스키)

#### 기본 상황 재확인
- 안양 자가 아파트, 남편(55, 중견기업 부장) / 성인 자녀 2명 독립
- 결혼 전 경리 5년 → 20년 전업주부. **근로 이력 사실상 전무**
- 용돈 월 50만원, 가계 여유 있음 → **돈이 동기 아님**
- 갱년기 진행, 무릎 시큰거림, 밤잠 설침
- 자녀 독립 후 허전함 → "나도 뭐라도 해야겠다" 생각 **반년째**
- 주 2회 동네 문화센터 요가 + 또래 엄마들 커피 — 유일한 사회 접점
- 친구 김 여사(동갑, 요양보호사 현직) = **유일한 롤모델 + 압박감 원천**
- 자격증 관련 구체 행동: 네이버에서 "요양보호사 학원" 검색 1회 → 창 닫음. 남편·친구 누구에게도 말 안 함

#### 핵심 인사이트 3개

**1. 자격증은 수단, 진짜 목적은 "소속감"과 "사회와의 재연결" ⭐⭐⭐**
- 학원 고르는 단 하나의 기준 질문에 → **"나랑 비슷한 사람이 많은 곳"**
- 본인 멘트: **"그 자격증 따러 가는 과정 자체가 필요한 사람이에요."**
- **"자격증보다 더 큰 거일 수도 있어요"** — 수단-목적 역전 인지
- 결정 프레임: "내가 여기 속할 수 있을까"
- → **김철수(생계)·박영수(커리어 전환)와 완전히 다른 축의 needs**
- → 땀앤땀스가 "자격증 비교"가 아닌 **"같은 처지 사람들 모여있는 곳"** 으로 포지셔닝하면 이 세그먼트 강력 흡수

**2. 실패 공포가 있지만 결이 다름 — "'최영희'로 평가받는 것"에 대한 공포 ⭐⭐⭐**
- 김철수(48, 남성) 실패 공포 = "150만원 날리면 끝" (경제적)
- 최영희 실패 공포 = **"20년간 엄마/아내였는데, '최영희'로 처음 평가받는 자리에서 못해내면 증명되는 게 무서움"** (정체성)
- 본인 자각 멘트: **"시작 안 하면 '안 해봐서 그래'라고 할 수 있잖아요. 해봤는데 안 되면 도망갈 데가 없어요."**
- 가족에게도 말 못함 — "공언했다가 못 따면 창피해서"
- 인터뷰 중 **"실패에 대한 두려움이 있으시군요"** 한마디에 눈물 — 본인도 몰랐던 이유
- → **"시작도 안 했는데 이미 자기를 실패한 사람 취급"** 자각 → 동기 급상승
- → 광고 카피 포인트: "실패해도 돌아올 곳"이 아니라 **"처음부터 내 자리가 있었던 것처럼"**

**3. "혼자 알아보고 혼자 겁먹는 반년" — 동반자 없는 의사결정 루프 ⭐⭐**
- 6개월간 혼자 고민, 아무에게도 말 안 함 (남편·친구 김 여사 포함)
- 학원 앞을 여러 번 지나감 → 들어가지 못함
- 본인 멘트: **"같이 가줄 사람이 없어요."**
- "분위기 좋다"는 **선행 정보만 있으면** 들어갈 수 있음 — 즉 **신뢰 가능한 안내 1명만 있으면 움직임**
- → 김철수·박영수도 공통으로 나온 **"누가 끌어줘야"** 패턴이지만, 최영희는 **발품 단계 이전에 "같이 가주기"까지 필요**
- → 컨시어지 서비스 needs 극명 (가장 강한 지불의사 후보)

#### 땀앤땀스 관점에서의 시사점

**A. 학원 상세 페이지 필수 정보 (최영희 세그먼트)**
- **현 수강생 연령대/성별 분포** (예: "50대 여성 65%, 40대 여성 20%…")
- **경력단절 입학생 비율** ("최근 3개월 입학생 중 전업주부 출신 18명")
- **수강생 브이로그/인터뷰 영상** (쿠팡 '사진 후기' 등가물)
- **강사 프로필에 "톤"** — 경력뿐 아니라 수업 스타일 영상 1분

**B. 컨시어지 기능 (지불의사 가장 높은 후보)**
- **"첫 방문 동행 신청"** — 상담 예약 + 당일 현장 동행 (실제 사람 아니어도 영상통화로 중계 가능)
- **내일배움카드 발급 대행** — 전업주부/경력단절 여성 자격 판별 및 고용센터 동행/원격 지원
- **"카톡으로 질문해주는 상담원"** — 고용센터 방문 전 예상 질문·답변 리허설

**C. 랜딩/광고 메시지 (최영희 트리거)**
- "같은 또래 50대 전업주부 ○○명이 작년에 자격증 땄어요"
- "요양보호사 학원, 혼자 가기 무서우시죠? 같이 가드릴게요"
- "자격증보다 친구가 먼저 생길 수도 있어요"
- "20년 만의 공부, 가장 편안한 분위기부터 찾아드려요"
- **"○○ 엄마가 먼저 간 그 학원"** — 친구 영향력을 카피로 변환
- **금지**: 합격률·취업률 강조 (최영희는 관심 없음. 차가워 보임)
- **금지**: 결제 압박/마감 임박 (박영수 거부감과 동일)

**D. 세그먼트별 랜딩 분리 필요 ⭐**
- 김철수형(실직 위기) ↔ 최영희형(사회 연결)은 **동일 랜딩으로 양쪽 잡기 불가**

2. **쿠팡 후기 필터링 안목은 매우 높음** — 정보 리터러시가 낮을 거라는 편견 오류. 학원 후기도 같은 수준의 신뢰 구조(사진/중립 후기/일반인 영상)로 제공하면 바로 해독함

5. **가족에게 말 못함** — 김철수는 아내에게, 최영희는 남편에게. **"말 안 한 채로 시작할 수 있는"** UX (카톡/폰 한 대로 완결) 필수

#### 핵심 인사이트 3개

**1. "자영업 폐업자"는 실업자도 재직자도 아닌 자격 사각지대 ⭐⭐⭐**
- 결정적 멘트: **"저는 실업자가 아니잖아요. 제가 사장이었으니까. 자영업자가 망한 건데. 이것도 해당이 되는 건지..."**
- 내일배움카드 = "실업자가 받는 것"이라는 고정관념 + 검색 결과 혼재("자영업자도 가능" vs "폐업신고 필요")
- → **김철수(이름만 앎)·박영수(발급 완료)·최영희(전업주부 자격 의심)와 다른 제4의 세그먼트**
- → 랜딩 카피 **"자영업 폐업하신 분도 됩니다"** 한 줄만 있으면 진입 가능

**2. 학원 광고 전화 트라우마 = 리드젠 UX 절대 금지 ⭐⭐⭐**
- 경험: 상담 신청 폼에 번호 입력 → 바로 전화 → 뭘 물어야 할지 몰라 "알겠습니다" 하고 끊음 → **계속 걸려옴 → 모르는 번호 전부 차단**
- 결과: **"이제 검색하다가 '무료상담' 버튼만 봐도 덮어요"**
- 학원 광고 생태계 자체에 대한 학습된 거부감
- → **땀앤땀스 UX 확정 금지사항: "번호 입력 → 학원이 전화" 플로우 절대 금물**
- → **"전화 안 받아도 정보 다 볼 수 있어요"** 자체가 차별점이 됨 (박영수의 "숫자 공개"와 다른 결의 차별점)

**3. 딱 하나의 기준은 "취업 실체" (같은 또래보다 상위) ⭐⭐⭐**
- 처음엔 "같은 또래 있는 곳"이라 답함 → 인터뷰어가 "딱 하나만" 압박
- 긴 침묵 후: **"취업이에요. 취업. 아까는 제가 부끄러워서 그런 거고, 근본적으로는 일하려고 따는 거니까요."**
- 구체 요구: **"작년에 우리 학원에서 몇 명이 어디 갔습니다. 특히 50대 아저씨가 어디 갔는지."**
- → 박영수의 "원자료 공개"와 동일하나, 최영희(관심 없음)와 완전히 대비
- → 땀앤땀스 학원 상세에 **"50대 합격자 취업처 실명"** 섹션 필요 (김철수·박영수·이정호 공통 니즈)

1. **"자영업 폐업"은 내일배움카드 인지 장벽의 숨은 거대 세그먼트** — 국세청 폐업 통계와 교차하면 TAM 산출 가능

#### 핵심 인사이트 3개

**1. "공부 트라우마" = 필기 있는 자격증 즉시 탈락 기준 ⭐⭐⭐**
- 전기기능사 관심 있었으나 **"필기 있다" 알자마자 즉시 접음**
- 본인 멘트: **"책만 봐도 속이 안 좋아요. 2년 동안 도서관에서 뭐 했나 싶고."**
- 몸 쓰는 일에 끌리는 근본 이유: **"하루 일하면 하루치가 남잖아요. 공부는 2년 해도 떨어지면 0이에요."**
- → "결과가 즉시 손에 잡히는 것"이 가치 판별 1순위
- → 진단/필터에 **"필기 시험 유무"** 옵션 노출이 이 세그먼트 결정적 필터
- → 박영수(체계적 비교)·이정호(취업 실체)와 다른 결의 **"공부=실패 재현"** 트라우마 세그먼트

**2. "마지막 카드" 공포 — 동기 최고치 × 행동 최저치 역설 ⭐⭐⭐**
- 인터뷰어 "그냥 관심 단계인가요?" 질문에 강하게 반박: **"아니요. 따고 싶어요. 진짜로."**
- 결정적 멘트: **"너무 하고 싶으니까 무서운 거예요. 이것도 안 되면 저 진짜 끝이니까."**
- 공무원 시절엔 "한 번 더 하면 된다" 여유 있었음 → 지금은 **"마지막 카드"** 인식
- 역설: 동기 = 4페르소나 중 최고, 행동 = 최저 (1달째 전화 0회)
- → 박영수(손실 회피 = 과거 자산 보호)와 다른 결: **"실패 적립으로 이미 바닥. 한 번 더 = 0 아래로"**
- → 광고 카피는 **"한 번 더"가 아니라 "처음이라 생각해도 돼요"** 계열
- → 땀앤땀스가 "작게 쪼개서 시작시키기"가 유효 (전체 등록 X, 전화 1건 X)

**3. "재촉 피로 vs 인정 감동" — 인터뷰 중 급속 전환 ⭐⭐⭐**
- 주변 반응: **"야 일단 시작해", "생각이 너무 많다", "행동해야지"** 반복 → 더 못 움직임
- 인터뷰어 "돈 벌러 가는 건데 신중해야죠" 한 마디에 **즉시 목소리 밝아짐**
- 본인 멘트: **"신중해도 되는 거였네요. 그동안 제가 너무 느린 거라고만 생각했는데."**
- 직후 자발적 행동 선언: **"오늘 집에 가서 기준 세우고, 두세 개만 전화해볼게요"** (처음엔 "10개 비교" 압도 → "두세 개는 가능")
- → 김철수·박영수·최영희·이정호 공통 **"누가 옆에서 짚어줘야"** 멘트의 정민재 버전: **"재촉 말고 인정하면서 짚어주기"**
- → 서비스 톤: **"느린 게 당연" 공감 후 "두세 개만 추려드릴게요"** (10개 비교 UX = 이탈 신호)

- **"28살, 경력 0, 이런 사람도 취업한 곳"** (동일 코호트 후기)
- **"전화 말고 카톡으로"** (이정호와 공통 UX)
- **금지**: "지금 시작하세요" "행동이 답" 류 재촉 카피 → 주변이 이미 다 한 말, 피로

**B. UX/기능 강조점**
- **"필기 시험 有/無" 필터** 진단 초기 질문 필수 (이 세그먼트 고유)
- **연령 코호트 후기** — "20대 후반 공시 포기자" 코호트 블록
- **"작게 쪼개기"** — 전체 등록 플로우 대신 "학원 2곳 연락처 + 예상 멘트 템플릿" 식 미니 전환
- **상담 톤**: 이력 먼저 묻지 말고 **"어떤 상황이세요?"**
- **마지막 카드 공포 완충**: 환불·중도 포기 시 불이익 투명 공개 (학원 측에 요구)

**C. 세그먼트 차별화 — 청년 실업 미수혜자**
- 정민재는 **실업급여 사각지대** (근로 이력 無, 공시생은 실업자 통계에 잡히지 않음)
- 내일배움카드 자격은 됨 (만 15세 이상 미취업자 적용) → **"공시 포기자도 됩니다"** 명시 필요
- 유튜브 "20대 백수 자격증 추천" 알고리즘이 유입 채널 → 이정호(50대 자격증)와 구분된 컨텐츠 카테고리
- 랜딩 분리 후보: **"공시/고시 접고 방향 못 잡은 20대"** 서브 랜딩 (자격증 이름 대신 상황이 제목)

#### 땀앤땀스 반응 (1~5점): **미측정** (컨셉 테스트 미진행)
→ 후속 세션에서 특히 **"필기 無 필터"**, **"코호트 후기"**, **"2~3개 큐레이션 결과"** 반응 측정 필요

#### 놀라운 발견 💡
1. **행정 처리 역량의 일시적 저하** — 2년간 공무원 시험 서류·원서·시험장 다 해낸 사람이 학원 전화 한 통 못 누름. **"한 번 무너지니까 작은 것도 다 무겁다"** → 장벽은 절차 난이도가 아니라 **"무너진 사람의 행동 심리"**. UX는 난이도 낮추기보다 **"여기서부터 시작하면 된다" 재출발 감각**이 핵심
2. **유튜브 알고리즘이 유일한 진로 상담사** — 친구·가족·동창·학교 취업센터 전무. **"너는 이거 해라고 정해줬으면"** 멘트 → 땀앤땀스의 "큐레이션" 기능이 감정적 위안까지 커버할 수 있음
3. **"한 번 더 실패 = 0 아래" 심리** — 박영수(손실 회피)와 구조적으로 반대. 박영수는 "내가 쌓은 것 지키려" 못 움직이고, 정민재는 **"이미 다 잃어서 더 잃을 게 없는데도" 못 움직임**. 자존감 바닥 세그먼트에는 **"작은 시작" UX**가 유일 해법
4. **"생각이 너무 많아서 못 시작한다" 자가진단이 행동 정지 루프를 강화** — 본인이 이 문장으로 자기를 설명하는 순간 다음 행동이 더 멈춤. 땀앤땀스 온보딩에서 **"생각 많은 게 당연해요" 인정 후 다음 선택지 하나만 노출**이 행동 촉진
5. **"재촉 → 차단, 인정 → 감동" 극명한 반응** — 주변 멘트 대부분 재촉형. **"신중해도 된다"** 한 문장에 즉시 **"오늘 두세 개 전화해볼게요"** 행동 선언. 단, 10개는 거부. **감정 인정 + 인지 부하 한계 존중**이 정민재 세그먼트 마케팅의 핵심
6. **"평생 지게차만 할 것" 천장 공포** — 자격증 취득·취업이 끝이 아니라 **"다음 계단"** 있어야 움직임. 땀앤땀스 2단계(일자리 매칭) 너머로 **"현장 경력 N년 후 이직·승급 경로"** 약한 시각화도 20대 세그먼트 진입 촉진 가능
7. **부모와 말 안 한 채로 시작하고 싶음** — 김철수(아내)·최영희(남편)·이정호(아내에 미안함) 공통. 정민재는 "아버지 눈 안 마주침"이 가장 구체 장면. **"가족 공언 없이 시작 가능한" UX** (카톡·폰 한 대 완결, 우편물 없음)가 4~5페르소나 공통 필수

#### 광고 카피 후보 (정민재가 반응할 말)
- **"공무원 접고 뭐 할지 모르겠을 때."** ⭐ (상황 정조준)
- **"'야 일단 해' 말고, '신중해도 된다'고 말해드려요."** ⭐ (감동 포인트 직격)
- **"학원 10개 말고, 딱 2개만 추려드려요."** ⭐ (인지 부하)
- **"필기 없는 자격증만."** (공부 트라우마)
- **"28살, 경력 0, 이력서 뭐 써야 할지 모를 때."** (또래 코호트)
- **"전화 대신 카톡으로 먼저."** (이정호와 공통)
- **"지게차 따고 3년 뒤, 그 사람은 지금 뭐 할까요?"** (천장 공포 해소)
- **"아버지 퇴직 전에, 딱 한 걸음만."** (감성 — A/B 대상, 리스키)

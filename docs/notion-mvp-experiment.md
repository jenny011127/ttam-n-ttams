## 📌 한 줄 요약

**내일배움카드로 수강 가능한 기피 자격증 시장에서, "적성진단 + 학원 큐레이션" 서비스가 CPA 모델로 성립할 수 있는지 2주간 검증한다.**

---

## 🧭 서비스 본질 (중요)

> **"내일배움카드는 있지만, 뭘 배울지 몰라서 시작 못 하는 사람을 움직이게 하는 것"**

### 핵심 원칙
- 내일배움카드 = **타겟 필터 ❌** / **행동 촉진제 ✅**
- 강남언니 모델(marketplace) ❌ / 토스 모델(decision companion) ✅
- 비교 플랫폼이 아니라 "결정 동반자"

### 경계할 것
- 학원 비교 기능 고도화 욕심 ("비교"는 본질 아님)
- 추천 정확도 집착 (완벽한 추천보다 "일단 움직이게")
- 광고 최적화 매몰 (CPA만 쫓다 타겟 희석 주의)
- 기능 추가 유혹

## 👥 타겟 페르소나 (AI 인터뷰 기반 5세그먼트)

| 페르소나 | 핵심 | 본질 장벽 | 광고 톤 |
|---------|------|----------|--------|
| 김철수 (48, 제조업 폐업 위기) | 생계 유지 | 실패 공포 (경제적) | "무료 / 대신 해드려요" |
| 박영수 (35, 영업 퇴사 준비) | 커리어 전환 | 자존심 / 손실 회피 | "숫자로 보여드립니다" |
| 최영희 (52, 전업주부 재취업) | 사회 재연결 | 정체성 + 소속감 | "같이 가드릴게요" |
| 이정호 (55, 자영업 폐업) | 생계 + 자존 회복 | 자격 불확실 + 광고 피로 | "자영업도 됩니다 / 전화 안 남기셔도" |
| 정민재 (28, 공무원 포기) | 출발선 서기 | "마지막 카드" 공포 | "10개 말고 2개만 / 신중해도 돼요" |

### 전원 공통 패턴 (중요 인사이트)
1. **"누가 다 알려주면 좋겠다"** — 5/5 전원
2. **"가족에게 말 못함"** — 5/5 전원 (→ 가족 몰래 완결 UX 필수)
3. **학원 광고/마케팅 불신** — 5/5 전원 (→ "원자료 공개"가 차별점)
4. **전화 공포** — 3/5 (이정호, 정민재, 최영희)
5. **내일배움카드 인지 장벽 5단계** — 전원 다른 형태로 존재

3. 지게차운전
4. 도배기능사
5. 타일기능사
6. 용접기능사

## 🧪 실험 가설

### 상위 가설
**"내일배움카드로 수강 가능한 기피 자격증 시장에서, 적성진단 + 학원 큐레이션 서비스가 CPA 모델로 성립할 수 있다."**

### 하위 가설 (Testable)

| # | 가설 | 측정 지표 | 성공 기준 |
|---|------|----------|----------|
| H1 | 수도권에 기피 자격증 관심자가 유의미하게 존재 | 광고 CTR, 랜딩 진입수 | CTR 1.5%+ |
| H2 | 30초 진단이 리드로 전환됨 | 진단 시작→완료→리드 | 완료 40%+, 리드 전환 30%+ |
| H3 | "내일배움카드 무료 수강" 메시지가 핵심 촉발제 | A/B CVR 비교 | 내일배움카드 강조 버전 우세 |
| H4 | 5개 페르소나별 반응 차이가 광고 카피로 검증됨 | UTM별 CPA | 세그먼트 간 CPA 2배+ 차이 |
| H5 | 학원 제안 가능한 CPA 수준 | 광고비 ÷ 유효 리드 | **CPA 2만원 이하** |
| H6 | "decision companion" 톤이 "학원 비교"보다 효과적 | 톤 A/B | 토스형 톤 우세 |

## 🎓 실험의 의의

### 왜 이 실험을 해야 하는가

1. **BM 성립 가능성 최초 검증** — 학원이 지불할 만한 CPA인지 돈으로 증명
2. **5개 페르소나 실측** — AI 인터뷰 인사이트 → 실제 유저 반응 cross-check
3. **포지셔닝 검증** — 강남언니 vs 토스, 어느 쪽이 먹히는지
4. **Phase 2 투자 결정 근거** — 허브 전면 개편 vs MVP 강화
5. **학원 영업 자료** — 실측 데이터로 제안

## 🛠️ UX 수정 결정사항 (실험 전 필수 작업)

### 1. 전화번호 수집 플로우 재배치
- **변경**: Collect 앞 → 뒤로 (결과 게이트)
- **플로우**: 랜딩 → 바로 진단 → 완료 → 결과 보기 전 **전화번호 필수 입력** → 결과 공개 → Next Action
- **효과**: 진단 이탈 감소 + 동기 최고조에서 전화 허들 넘음

### 2. 히어로 = 구체 숫자 기반 메시징
- **금지**: "합격률 1위", "업계 최고", "만족도 98%", "지금 할인"
- **권장**:
  - 메인: "내일배움카드로 무료 수강, 뭘 배울지 모르겠다면?"
  - 서브: "30초 테스트로 딱 2~3개만 추려드려요"
  - 숫자 예시: "이 달 ○○○명이 테스트 받았어요"

### 3. 내일배움카드 자격 FAQ — 유쾌한 포용 톤
```
누가 올 수 있냐고요?

재직자 OK. 전업주부 OK. 자영업 폐업하신 분 OK.
공시 끝내신 분도, 고졸 취업자도, 쉬었음 청년도.
AI에 맞설 사람, 다 환영합니다.
```
- **브랜드 시그니처**: 다른 국비지원 플랫폼과 차별화되는 톤
- **포함 대상**: 재직자 / 전업주부 / 자영업 폐업자 / 공시 포기자 / 고졸 취업자 / 쉬었음 청년 / AI에 맞설 사람

### 4. 이질감 제거 — 랜딩↔웹앱 매끄러운 연결
- **시각적 일관성**: 디자인 토큰 공유, 페이지 로딩 화면 삭제
- **스토리 연속성**: 결과 헤더가 웹앱에 그대로 이어짐
- **CTA 호기심 유발형**: "신청하기" ❌ → "내 학원 보러 가기" ✅
- **선택지 최소화**: 화면당 CTA 1개, sticky CTA 유지
- **마이크로 인터랙션**: 살짝 흔들리는 버튼, 화살표 이동, 스크롤 힌트

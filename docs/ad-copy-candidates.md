# 광고 소재 카피 후보 (2026-04-23)

> Meta 광고 이미지/영상 소재용 카피 저장소. 랜딩 히어로와는 별도 관리.
> 랜딩과 톤 매칭 + 클릭 유도 목적.

---

## 🎣 훅 (Hook) 카피

### 의문형 자극 ⭐ (유튜브 썸네일 공식)
```
"한 달 500 버는 기술직,
50대도 3개월이면 된다고?"
```
- 타겟: 김철수(48), 이정호(55), 정민재(28 이하는 청년 버전 따로)
- 강점: 월수입 + 기간 + 연령으로 즉각 호기심
- 변형: "40대도 3개월이면?", "주부도 한 달이면?"

### 위기감 + 대안 (공포 → 호기심)
```
"공장 문 닫기 전에,
 옆 사람은 벌써 자격증 따서 나갔대요"
```
- 타겟: 김철수, 이정호
- 강점: 불안한 현재 + FOMO
- 변형: "회사가 흔들릴 때, 먼저 움직인 사람은 뭘 했을까?"

### 성공 사례 나열 (썸네일형)
```
"48세 공장 김 씨 → 지게차 월 380
 52세 주부 이 씨 → 요양보호사 월 220
 35세 영업 박 씨 → 용접 월 500"
 
 "당신 차례는?"
```
- 타겟: 전 페르소나
- 강점: 구체성, 소속감
- 변형: 연령·직업 조합 다양화

```

 들어봤어도 시도는 안 해봤죠?"
```

### "왜"류
```

 이 자격증으로 몰려갈까?"
```

### "비밀"류

```
 안 알려주는 학원 고르는 법"
```

## 💰 현실 타겟 카피

### 수익 소구 (직설)
```
"자격증 하나 바꿨더니
 월수입이 2배."
```

### 전환 소구 (직업 변경)
```
"영업 10년, 이제 몸 편한 기술직으로."
"주부 20년, 다시 월급 받는 법."
```
- 타겟: 박영수, 최영희

### 공감 소구 (감정)
```
"'야 일단 해' 재촉받으셨죠?
 신중해도 된다고 말해드릴게요."
```
- 타겟: 정민재
- 인터뷰 감동 포인트 활용

## 🎯 페르소나별 저격 카피

### 김철수 (48, 제조업 폐업 위기)
- "공장 15년, 다음 15년은 뭐로 벌어요?"
- "48에 지게차 따면 늦을까요? 48에 딴 김 씨는 지금 월 380."

### 박영수 (35, 영업 퇴사 준비)
- "2년째 '알아만' 보고 계신가요?"
- "35에 용접? 늦은 줄 알았는데, 박 씨는 지금 월 500."

### 최영희 (52, 전업주부 재취업)
- "'OO 엄마는 땄대요' 6개월째 들으셨죠?"
- "50대 여성이 가장 많이 가는 학원, 알려드려요."

### 이정호 (55, 자영업 폐업)
- "치킨집 접고 다음엔 뭐 할까요?"
- "자영업 폐업해도 국비지원 되는 거 알고 계셨어요?"

### 정민재 (28, 공무원 포기)
- "공무원 포기했는데, 뭐부터 해야 할지 모르겠다면"
- "10개 비교 말고, 딱 2개만 추려드릴게요"

## 🚫 피해야 할 카피 패턴

### 과장/자극 방향
- ❌ "누구나 월 1000 가능" (의심)
- ❌ "3일이면 끝" (비현실)

### 일반 마케팅 태
- ❌ "합격률 1위" (검증 불가)
- ❌ "지금 할인" (영업 거부감)
- ❌ "업계 최고" (공허)

### 재촉
- ❌ "지금 시작하세요" (정민재 피로)
- ❌ "행동이 답" (훈계조)

## 📐 카피 체크리스트

새 카피 쓸 때마다 확인:
1. **누구 얘기인지** (페르소나 힌트)
2. **무엇을 받는지** (결과물 또는 약속)
3. **얼마 걸리는지** (시간 명시)
4. **얼마 드는지** (무료 or 숫자)
5. **다음 행동이 뭔지** (CTA 명확)

## 🖼️ 이미지/영상 소재 가이드

### Meta 피드 이미지 (1:1)
- 상단: 훅 카피 1줄 (큰 글씨)
- 중앙: 실명+직업+월급 3명 나열 or 성공 스토리 1명
- 하단: "30초 테스트" + URL 표시

### Meta 릴스/스토리 (9:16)
- 0~1초: 훅 카피 확 꽂기 (예: "월 500 버는 기술직?")
- 1~3초: 인물 비주얼 + 대사
- 3~5초: "30초 테스트" CTA
- 변동: 팬터(pantomime) 스타일 효과음 OK

### 유의사항
- 이미지 텍스트 비율 20% 이하 (Meta 정책)
- 페이스 이미지 사용 시 동의 or AI 생성
- "확실히 월 500 번다"는 단언 금지 → "벌어요" 정도 완곡
